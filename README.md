# agreGator

This program requires postgres and 
go to run, to instal the gator CLI, 
the commands are:
go install github.com/mf-stuart/gator

1. login \<username>
2. register \<username>
3. reset
4. users
5. agg \<time with units>
6. addfeed \<name> \<url>
7. feeds
8. follow \<url>
9. unfollow \<url>
10. following
11. browse \[limit]
